# Secure-Password-Tool 🔐

A responsive web-based password generator built with **HTML5**, **CSS3**, and **Vanilla JavaScript**.

## 🚀 Features
* **Custom Length:** Generate passwords of any length.
* **Security Options:** Include uppercase, lowercase, numbers, and symbols.
* **Responsive Design:** Works perfectly on both desktop and mobile devices.
* **One-Click Copy:** Easily copy the generated password to your clipboard.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3
* **Logic:** Vanilla JavaScript (ES6+)

## 📂 How to Run
1. Clone the repository.
2. Open `index.html` in any modern web browser.
